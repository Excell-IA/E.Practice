## Endpoint backend disponibili (36 route)

Dump completo in `frontend/openapi.json`. Sintesi blocchi:

- **Sessione/utenti**: `POST /api/session`, `GET /api/users`
- **Clienti**: `GET /api/clients` (paginato + filtri), `GET /api/clients/{id}`,
  `POST`, `PUT`, `DELETE` (soft, 409 se ha pratiche aperte),
  `GET /api/clients/{id}/practices`, **`GET /api/clients/search?q=`** (combobox enriched)
- **Categorie**: `GET /api/categories`
- **Templates**: `GET /api/templates/{cat_id}` (lista fasi),
  **`GET /api/templates/category/{cat_id}/preview?apertura=YYYY-MM-DD`** (per modal step 2/3)
- **Pratiche**: `GET /api/practices` (lista paginata), `GET /api/practices/{id}` (detail
  enriched: client+category+responsible+labels+phases+events+notes+attachments+collaborators+counts+progress_pct),
  `GET /api/practices/{id}/phases`, `GET /api/practices/{id}/events`,
  `GET /api/practices/{id}/labels`, `POST /api/practices` (con
  `create_default_reminders` + `label_ids`), `POST /api/practices/{id}/archive`
- **Fasi**: `PUT /api/phases/{id}` (solo pending), `POST /complete` (+ reminder
  opzionale), `POST /skip`
- **Eventi**: `POST /api/events`
- **Note**: `POST /api/notes`
- **Allegati**: `GET /api/attachments` (upload reale a F22 R2)
- **Reminders**: `GET /api/reminders`
- **Activity log**: `GET /api/activity` (paginato + filtri)
- **Dashboard** (enriched): `GET /api/dashboard` (vedi sotto)
- **Search globale**: `GET /api/search?q=` (multi-entità)
